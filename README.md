# Studio Café
Landing page moderna para uma cafeteria, desenvolvida em React com Tailwind CSS e Framer Motion. O projeto conta com um carrossel automático de produtos, animações fluidas e um cardápio completo

Projeto criado em parceria com o designer Lucas Rodrigues

---

## Estrutura do Projeto

Navbar.jsx: Menu de navegação fixo

HeroCarousel.jsx: Banner principal com troca automática de produtos e animações de entrada/saída

Menu.jsx: Cardápio completo dividido por categorias (Cafés, Pães, Porções )

MainProducts.jsx: Grade com os produtos favoritos

FeaturedProducts.jsx: Seção de destaque com imagem e texto lateral

## Funcionalidades

Carrossel Automático: Muda o produto a cada 5 segundos sem necessidade de clique

Animações Fluidas: Elementos que flutuam (grãos e café) e transições suaves entre seções

Design Responsivo: Adaptado para dispositivos móveis e desktop
